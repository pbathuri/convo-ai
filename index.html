Convo AI

The full app is a Streamlit server. Host it on any Docker-capable platform (this repo includes a Dockerfile). Vercel only serves this static page; it does not run Streamlit.

Deploy the full app (cloud)

  • Render — In the dashboard: New → Blueprint → connect this repo (uses render.yaml). Set secret OPENAI_API_KEY. Health check: /_stcore/health.
  • Railway — New Project → Deploy from repo; Railway picks Dockerfile / railway.json. Add variable OPENAI_API_KEY.
  • Fly.io — Install flyctl, run fly launch (adjust app name in fly.toml), fly secrets set OPENAI_API_KEY=sk-..., fly deploy.
  • Google Cloud Run — gcloud run deploy --source . (or build the Dockerfile). Set OPENAI_API_KEY as an env var.

On the cloud, use OpenAI (OPENAI_API_KEY) unless you run a private Ollama instance on the same network. Local-only: USE_OLLAMA=true in .env — see .env.example.

Optional redirect

After you have a public Streamlit URL (e.g. https://convo-ai.onrender.com), set window.CONVO_AI_STREAMLIT_URL in the script below so this page forwards visitors.
